# MaterialRecord 耗材紀錄系統


## 說明
最初學習C#練習做的簡易系統，使用Asp.Net WebForm (.Net Framework)進行開發，語法/架構有空閒時後續能再做優化


## 系統介紹 ── Description
系統可針對公司企業軟體未控管的耗材或是外部廠商提供的物品進行進出廠區與使用紀錄的統整


## 開發工具及技術
- 技術：C#, JavaScript, HTML, CSS, Ajax, JQuery, MSSQL
- 工具：Visual Studio, MSSQL
